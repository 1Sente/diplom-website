# План разработки корпоративного сайта Web-студии и Хостинга (ИП)

Данный план предназначен для прохождения производственной и преддипломной практики. Проект: создание корпоративного веб-сайта для собственного ИП, занимающегося веб-разработкой и предоставлением услуг хостинга.

## Цели проекта
1. Создать презентабельный и быстрый сайт для привлечения клиентов.
2. Продемонстрировать навыки полного цикла веб-разработки (Full-stack).
3. Реализовать панель администратора для управления контентом и заявками.
4. Спроектировать базу данных для хранения услуг, тарифов хостинга, портфолио и заявок клиентов.

## Рекомендуемый технологический стек (на выбор)

**Вариант 1: Быстрая разработка с мощной админкой (Рекомендуемый для ИП)**
* **Backend & Admin:** Python + Django (предоставляет готовую админ-панель "из коробки" и надежную ORM).
* **Frontend:** Шаблоны Django + Bootstrap/Tailwind CSS или отдельный Frontend на Vue/React.
* **База данных:** PostgreSQL.

**Вариант 2: Современный JavaScript/TypeScript стек**
* **Frontend:** Next.js (React) + Tailwind CSS.
* **Backend & Admin:** Payload CMS или Strapi (Headless CMS с отличной админкой).
* **База данных:** PostgreSQL или MongoDB.

## Архитектура Базы Данных (Минимальная)
1. **Услуги (Services):** Название, Описание, Цена (от), Иконка/Изображение.
2. **Тарифы хостинга (Hosting Plans):** Название, Объем диска, Трафик, Базы данных, Цена.
3. **Портфолио (Projects):** Название, Описание, Ссылка на проект, Изображения, Используемые технологии.
4. **Заявки (Leads/Requests):** Имя, Контакты, Описание задачи, Статус (Новая, В работе, Завершена), Дата создания.
5. **Пользователи (Users):** Администратор (суперпользователь).
