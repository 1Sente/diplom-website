# План разработки корпоративного сайта Web-студии и Хостинга (ИП)

Данный план предназначен для прохождения производственной и преддипломной практики. Проект: создание корпоративного веб-сайта для собственного ИП, занимающегося веб-разработкой и предоставлением автоматизированных услуг хостинга (IaaS/PaaS).

## Цели проекта
1. Создать презентабельный и быстрый сайт для привлечения клиентов.
2. Продемонстрировать навыки полного цикла веб-разработки (Full-stack), включая системное администрирование и работу с API гипервизоров.
3. Разработать Личный Кабинет (ЛК) клиента для управления приобретенными услугами.
4. Автоматизировать выдачу виртуальных серверов через интеграцию с Proxmox VE API.
5. Реализовать панель администратора (Payload CMS) для управления контентом, пулом IP-адресов и заявками.

## Технологический стек (Утвержденный)
* **Фронтенд & Бэкенд:** Next.js (App Router, React) + TypeScript + Tailwind CSS.
* **Админ-панель & База Данных:** Payload CMS 3.x (Headless CMS интегрированная с Next.js) + PostgreSQL.
* **Инфраструктура:** Личный сервер HPE DL360 Gen9 под управлением Proxmox VE.
* **Автоматизация:** Proxmox REST API, Cloud-Init (для настройки ОС и установки панелей управления типа FastPanel/HestiaCP при клонировании шаблона).
* **Эквайринг:** YooKassa (ЮKassa) для приема платежей.

## Архитектура Базы Данных (Payload CMS Collections)
1. **Customers (Клиенты):** Пользователи Личного Кабинета (email, хеш пароля, профиль).
2. **Hosting Plans (Тарифы хостинга):** Название, ID шаблона в Proxmox (vmid), CPU, RAM, Disk, Цена.
3. **IP Pool (Пул IP-адресов):** IP-адрес, Статус (Свободен/Занят), Привязанная услуга.
4. **Hosting Instances (Купленные хостинги):** Связь с клиентом, Связь с тарифом, Выданный IP, Сгенерированный пароль, Статус (Создается/Активен/Остановлен), VMID в Proxmox.
5. **Orders (Заказы):** История транзакций из ЮKassa, Статус оплаты.
6. **Services & Projects (Сайт):** Услуги веб-разработки и Портфолио для публичной части сайта.
7. **Leads (Заявки):** Формы обратной связи (Заказать сайт).
